# Data Included in This Repository

This folder is for small, public-safe project data.

Raw GoPro videos and generated demo videos are intentionally excluded from git because they are large and contain private scene footage. The local copies remain under `runs/`.

Included data can contain:

- annotation CSV files used to train the lightweight navigation decision classifier
- calibration summaries for the GoPro/ArUco distance experiment
- small metadata files that help reproduce the reported metrics

If you need to reproduce the full report, keep the raw videos locally or publish selected assets separately as release files.
